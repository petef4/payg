STL test suite

This is a set of 225 tests to check conformance of C++ compilers and libraries to the ISO Standard. The specific area covered is STL.

This testsuite is derived from the STLport testsuite, which is derived from the Cygnus Solutions STL testsuite, which is based on ObjectSpace STL examples. Unlike the STLport version, STLport is not needed to run this.

stltest-1.00.tar.gz (83601 bytes)

Installation and running

This requires only modest know-how. Basically you unpack it, copy (and edit) a suitable .mak to Makefile, and then

make check

A perfect compiler would complete the checks. For each failure of foo1 a manual skip is required.

make dummy FUNC=foo1

That might be automated in a future version.

The result of make check is the file stl_test.out. Compare that with the expected results in stl_test.exp. See the end of README.txt for some expected differences, especially those due to different random number generators.
