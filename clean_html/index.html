HTML and XHTML example files

These are small examples of various aspects of HTML.

Examples of documents

bare.html
Absolutely minimal HTML document.
bare.xhtml
Absolutely minimal XHTML document.
standard.html
HTML document with the recommended minimum structure.
standard.xhtml
XHTML document with the recommended minimum structure.
standardx.html
XHTML document to be served as text/html with the recommended minimum structure.

Tests of HTML features

ConditionalComment.html
Testing Microsoft Conditional Comments
empty_strict.html
Empty XHTML element in HTML 4.01 strict. The document does not validate but it does render okay in most browsers.
empty_transitional.html
Empty XHTML element in HTML 4.01 transitional. The document does validate and renders okay in most browsers. An SGML compliant UA should render an extra >.
twometa_transitional.html
Test of two meta elements with transitional doctype.
